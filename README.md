# Daily Task Board

A lightweight desktop app for managing daily tasks/notes and tracking small projects.

## Features
- Calendar-based tasks + notes
- Status workflow: Unfinished / Working / Postponed / Finished
- Projects dashboard (Study/Research/Lab/Coding/Personal)
- Reminders (desktop notifications) for tasks and projects
- Drag & drop scheduling + Today list ordering
- Dashboard analytics
- Export to PDF and print
- Excel (.xlsx) storage with color formatting
- Automatic daily backups + restore

## Run

```bash
pip install -r requirements.txt
python main.py
```

## Data files
The app saves to an Excel workbook (`.xlsx`). You can set a default location (e.g., OneDrive) using **Settings**.

## User manual
The manual is available inside the app via **Help**, and as a web page in `docs/` for GitHub Pages.
